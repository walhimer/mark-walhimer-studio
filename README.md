# Mark Walhimer Studio
## Generative Art & Design System v2.1 (2026)

This repository contains the professional generative art portfolio and centralized design system for Mark Walhimer.

### 📂 Project Structure
- **/shared-assets**: Centralized `tokens.css` and JavaScript engines (`dna.js`, `ui-logic.js`).
- **/art**: Standalone generative artworks refactored for system compliance.
  - **Bloom / Release**: Seeded DNA lifecycle study.
  - **Traveling Landscape**: Seeded terrain generation.
  - **Surrender Machines**: Hybrid emotional mechanics.
  - **Shared Ground**: Relational social geometry.
  - **Living Commons**: Relational co-presence study.

### 🛠 Technical Stack
- **Language**: Vanilla JavaScript (ES6+), HTML5, CSS3.
- **Libraries**: p5.js (1.11.11), Tone.js (14.7.77).
- **Design System**: Token-based architecture with HSB color logic and Emergent DNA seeding.
